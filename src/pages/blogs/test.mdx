## Syntax highlighting

This example is using MDX page routing built into Astro while styled by TailwindCSS and Svelte components using daisyUI as a base to create interactive code highlighting.

For example, the following code snippet with component result are created by combining daisyUI's window mockup component with syntax highlighting with prose in markdown. Underneath, the result is rendered as a Svelte component using MDX.

<div class="mockup-window border bg-base-300">
  <div class="flex flex-col justify-center bg-base-200">
    ```svelte
      <script>
        export let count = 1;

        function incrementCount() {
          count += 1;
        }
      </script>

      <button class="btn block my-2 not-prose" on:click={incrementCount}>
        Clicked {count} {count === 1 ? 'time' : 'times'}
      </button>
    ```
    <div class="divider my-0 py-0" />
    <div class="text-center mx-auto py-0 mb-16">
      ## Result
      <Counter client:visible />
    
    </div>
  </div>
</div>
